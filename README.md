# dvbapiNET

**dvbapiNET** ist ein Plugin für den DVBViewer oder MDAPI(+) kompatible Anwendungen.
Es erlaubt die OScam dvbapi-Schnittstelle in den Anwendungen zu verwenden.

Dieses Plugin ist ein reines Forschungsprojekt. Es ist nicht für die reguläre Nutzung vorgesehen.

## Kompilierung
Die Kompilierung erfordert das SDK für .NET. Hier wird für die Native Exports ildasm benötigt.
DllExports erstellt damit eine Native DLL welche dann die dotnet-Komponenten lädt.
